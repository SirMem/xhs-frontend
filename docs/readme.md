## XHS 前端（飞书多维表格插件）功能说明

该目录用于简单说明 `xhs/` 下的前端插件能力。前端基于 **React + TypeScript + Vite**，UI 使用 **Semi UI**，并通过 **@lark-base-open/js-sdk** 与飞书多维表格（Bitable/Base）交互。

### 功能概览（按 Tab）

#### 1) 详情抓取回填
- 选择“链接所在列”（URL 字段）以及需要回填的目标字段（标题/博主昵称/描述/点赞数/发布时间等）。
- 点击“开始抓取选中行”后，前端调用后端接口抓取笔记详情，并回写到当前表格的对应字段。
- 内置 Console Output 日志面板，便于查看抓取进度与错误信息。
- 支持在前端保存并复用小红书 Cookie（本地持久化），减少重复粘贴。

#### 2) 时间段统计
- 输入关键词，选择开始/结束时间（弹出时间选择器）。
- 选择完成后会自动转为 **13 位毫秒时间戳**，点击“开始统计”后提交到后端进行近似统计。
- 统计成功后以弹窗形式展示结果，并将返回字段转成中文说明（同时保留原始 JSON 便于排查）。

#### 3) 违禁词检测
- 支持直接输入待检测文本，或输入小红书笔记 URL（可选抓取标题/正文后再检测）。
- “静态词库优先”检测：基于本地词典命中给出类别、严重度、证据、改写建议等。
- 可选 AI 审核：前端仅提供“是否启用”开关，AI 的 Base URL / Model / API Key 由**服务端 `.env`** 配置读取（用户无需填写）。

#### 4) 实时监控
- 添加监控项：输入笔记 URL，可配置点赞/评论增量阈值、检查间隔、是否初始化 baseline 等。
- 监控列表：通过“查看监控列表”按钮打开弹窗展示最近监控项（服务端限制最多 200 条）。
- 支持对单条监控执行：立即检查、重置 baseline、启用/暂停、删除。

### 开发/构建

在 `xhs/` 目录下执行：

```bash
npm run dev
npm run build
```
